# Kørebog – Fælles bil

En lille hjemmeside til at føre regnskab over en fælles bil for Patrick, Marcus, Oliver og Kathrine.

- **Kørselslog:** vælg dato + person og skriv slut-km. Start-km føres automatisk videre fra sidste tur. Kørte km og beløb (km × pris, standard 1 kr/km) regnes ud.
- **Startkilometer:** bilens udgangspunkt er sat til 75502 (kan ændres i feltet øverst). Odometeret viser dette tal, indtil den første tur er tilføjet.
- **Afregning:** samlet km, beløb, betalt og "Rest at betale" pr. person.
- **Betalinger til far:** registrér MobilePay-betalinger, som trækkes fra i afregningen.
- Fungerer på mobil og computer. Data gemmes lokalt i browseren.
